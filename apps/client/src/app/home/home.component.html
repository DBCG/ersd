This site provides access to the eRSD specifications for electronic Case Reporting (eCR). The eRSD is used to guide triggering and reporting of initial case reports from Electronic Health Records. For more information, please visit the EHR triggering documentation for EHR implementers. The site may also contain specifications for other public health reporting programs and pilots.

New Versions of eRSD Specification
If you do not know which eRSD version to use: please contact your EHR vendor or for eCR Now App implementations you can also read the “eRSD Configuration Instructions” file in your eCR Now App download folder.

An eRSD Specification bundle contains the current Reportable Condition Trigger Codes (RCTC) list, reporting timing, reporting queries, and a computable representation of the reporting workflow. Each of the eRSD versions available contains the same RCTC.

Preview Upcoming eRSD Updates
New versions of the eRSD are under development: implementers may view in-progress drafts of upcoming eRSD Versions to anticipate possible structural changes that may affect queries.
Please note: Previews do not contain new trigger codes. The current trigger codes are only available upon full release in an eRSD production version.
